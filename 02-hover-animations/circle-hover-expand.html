Full-Width Reveals

Circle Mask Transitions

Hover anywhere to reveal a hidden layer that radiates outward from your exact cursor position using CSS clip-path.

🎯

Radial Unveil

The circle expands from precisely where your cursor enters, creating a natural, intuitive reveal that connects the user's action directly to the visual feedback. No animation frame needed — pure CSS clip-path transitions handle the heavy lifting.

Cursor-Driven Masks

Each reveal follows your mouse inside the element, making the interaction feel personal and direct.

🖱️

Position Aware

Using CSS custom properties (--mx, --my) updated by JavaScript mousemove events, the clip-path circle origin follows the cursor in real-time. This creates a magnetic, responsive feel that makes the interface feel alive.
