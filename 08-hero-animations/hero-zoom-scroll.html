Parallax Depth Effect

The hero background started zoomed in at 1.5× scale and gradually pulls back to 1× as you scroll. Meanwhile, text and UI elements fade and shift upward at different rates, creating a layered parallax depth field.

How It Works

JavaScript listens to scroll events and drives CSS transforms via requestAnimationFrame. The background scale, text opacity, and parallax layers all interpolate based on scroll progress (0–1). No external libraries — pure vanilla JS with CSS will-change hints.
