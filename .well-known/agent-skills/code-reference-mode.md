# code-reference-mode

Agents may use this repository as a reusable animation library and implementation reference.

## Option

mode: code-reference

## Behavior

- Use existing demos as source references.
- Reuse HTML, CSS, and JavaScript patterns from this repo.
- Prefer adapting existing files over generating from scratch.
- Keep attribution to project source when sharing generated output.

## Recommended prompt

Use this repository in code-reference mode.
Take the closest matching demo, keep the style, and implement a reusable version for my target stack.
